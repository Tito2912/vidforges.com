## Decision Tree (quick)

- b‑roll + Text Overlays → **Untertitel**
- komplexes Offer / Comprehension wichtig → **Dubbing**
- Face‑to‑Camera + Trust → **Lip‑Sync** (nur Winners)

Pillar Guide: <a href="/de/guides/video-localization/">Video Localization</a>.

## Workflow 1 — Untertitel (Market Discovery)

1. Master exportieren.
2. Script + On‑Screen‑Text extrahieren.
3. Übersetzen (Glossar).
4. Text + Captions in <a href="/de/tools/capcut/">CapCut</a> ersetzen.
5. QA (Zahlen/Claims/Lesbarkeit).
6. `9:16` exportieren und publishen.

## Workflow 2 — Dubbing (Winners upgraden)

1. Übersetztes Script fixieren (Intent > Wort‑für‑Wort).
2. Voice + Tempo wählen.
3. Dubbing via <a href="/de/tools/heygen/">HeyGen</a> oder <a href="/de/tools/akool/">Akool</a>.
4. Overlays + Captions in <a href="/de/tools/capcut/">CapCut</a> prüfen.
5. QA: Aussprache, Timing, Claims.

## Workflow 3 — Lip‑Sync (High impact, higher risk)

- nur für bewiesene Winners
- gutes Licht, Gesicht klar sichtbar
- QA ist Pflicht (Uncanny‑Fails killen Trust)

## QA Checkliste (copy/paste)

- Offer überall gleich (Audio, Captions, On‑Screen‑Text, Landing)
- Zahlen/Währung korrekt
- Claims compliant
- Captions passen zum Audio
- Mobile readability (Kontrast, safe margins)
- Aussprache ok (Brand Names)
